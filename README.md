# Reporting-Ops
An operational hub bridging raw data and workforce performance. This repo tracks 1,400+ daily orders, 80%+ SLA compliance, and dark store costs across India. By merging Myntra hourly projections with real-time rider tracking and "Deep Pain" analytics, it transforms complex logistics into actionable, human-centric insights.
